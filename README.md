# Acme Dashboard — Next.js App Router

A full-stack dashboard application built while completing the **Next.js App Router Fundamentals** course by Vercel.

**Live Demo:** [proyecto-next-beta.vercel.app](https://proyecto-next-beta.vercel.app)

---

##  Features

- Authentication with **NextAuth.js** (Credentials)
- Dashboard with invoices, customers and revenue stats
- PostgreSQL database integration
- Search and pagination
- Form validation with **Zod**
- Fully responsive with **Tailwind CSS**
- Server Components & Streaming with **Next.js 14**

---

##  Tech Stack

| Technology | Usage |
|---|---|
| Next.js 14 | App Router, Server Components |
| TypeScript | Type safety |
| PostgreSQL | Database |
| NextAuth.js | Authentication |
| Tailwind CSS | Styling |
| Zod | Schema validation |

---

##  Getting Started

### Prerequisites
- Node.js 18+
- pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/laujml/proyecto-next.git
cd proyecto-next

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env
# Fill in your .env with your PostgreSQL and NextAuth credentials

# Run the development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
